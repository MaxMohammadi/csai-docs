# Data Aquisition

The Data Aquisition team creates and maintains a library of web scrapers the periodically update our knowledge base. Their work allows Nimbus to answer questions like "Who is teaching CSC 357 this quarter?". These developers familiarize themselves with both the arcitecture of the web and the Nimbus database in order to effectively collect and store useful information.

## Contributions

- Design web scrapers in Python to collect information on topics of interest
- Build data pipelines to integrate new information into the Nimbus model
- Maintain the quality of the collected data corpus
